# John Doe - Web Developer Portfolio

A modern, fully responsive portfolio website built with **HTML, CSS, and JavaScript**.  
Showcases skills, projects, and includes a **contact form** powered by **EmailJS**.  

## Features
- Home, About, Portfolio, and Contact pages  
- Mobile-friendly hamburger menu  
- Skills & portfolio sections with card layouts  
- Gradient red theme with CSS variables  
- Clean, organized, and easy-to-customize code  

## Installation
1. Clone the repo: `git clone https://github.com/yourusername/portfolio.git`  
2. Open `index.html` in your browser  
3. For the contact form, replace EmailJS keys in `contact.html`
